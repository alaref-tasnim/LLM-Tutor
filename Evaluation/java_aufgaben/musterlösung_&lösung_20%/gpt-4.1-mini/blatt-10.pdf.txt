public class HashTableChaining<K, V> implements HashTable<K,V> {

    @SuppressWarnings("unchecked")
    public HashTableChaining() {
        this.array = new ArrayList[SIZE];
    }

      @Override
    public void insert(K key, V value) {

            throw new NullPointerException("value must not be null");
        }
        int h = hashValue(key);

            l = new ArrayList<>();

                    e.setValue(value);

            }

            l.add(0, entry);
        }
    }

### (a) Funktion `insert(K key, V value)`

**Fehler und Probleme:**

1. **Zeile 21:** In der `if (l == null)`-Verzweigung wird das neu erstellte ArrayList `l` initialisiert und ein neues `HashEntry` hinzugefügt, jedoch wird `l` nicht wieder im `array[h]` gespeichert.
